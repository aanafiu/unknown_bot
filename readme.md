# 📖 Overview

UnknownBot is an AI-powered Discord bot that helps users improve their English naturally and professionally.

Instead of only correcting grammar, UnknownBot analyzes the user's writing and provides:

- ✅ Grammar correction
- 🌍 Native English rewrites
- 💼 Professional communication
- 🎯 Better sentence expressions
- 🇧🇩 Bangla translation
- 📊 IELTS writing evaluation
- 💡 Personalized learning tips

It is powered by modern Large Language Models (LLMs) through **OpenRouter**.

---

# ✨ Features

## 📝 Grammar Check

Detects grammatical mistakes and explains them clearly.

---

## ✨ Corrected Version

Rewrites your sentence while preserving its original meaning.

---

## 🌍 Native English

Shows how a native English speaker would naturally express the same idea.

---

## 💼 Professional Writing

Converts your text into a professional business or technical communication style.

Perfect for:

- Emails
- Job Applications
- Customer Support
- Technical Documentation

---

## 🎯 Better Expressions

Provides three alternative versions:

- Casual
- Neutral
- Formal

---

## 🇧🇩 Bangla Translation

Explains the professional version in natural Bangla.

---

## 📊 IELTS Feedback

Receive an estimated IELTS writing score with:

- Overall Band
- Grammar Score
- Vocabulary Score
- Fluency Score
- Improvement Suggestions

---

## 💡 Learning Tips

Every response includes practical tips to improve your English over time.

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/your-username/UnknownBot.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
discord_token_id=YOUR_DISCORD_BOT_TOKEN

OPENAI_API_KEY=YOUR_OPENROUTER_KEY

OPENAI_BASE_URL=https://openrouter.ai/api/v1
```

Run the bot

```bash
python bot.py
```
